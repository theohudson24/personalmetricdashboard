# Support

Personal Metric Dashboard is currently a private beta.

- Testers should use the in-application **Report a bug** feature for ordinary product issues.
- Repository contributors may use the GitHub issue forms for reproducible engineering work.
- Security concerns and suspected data exposure must follow [SECURITY.md](SECURITY.md) and should not be posted publicly.

Support is currently provided on a best-effort basis by the repository owner. Do not include passwords, authentication links, database credentials, exported files, or private health information in a support request.
