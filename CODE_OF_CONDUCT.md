# Code of Conduct

## Standard

Contributors and testers are expected to communicate respectfully, protect private user information, and provide feedback focused on improving the application.

Unacceptable behavior includes harassment, publishing another person's private information, sharing credentials, or intentionally attempting to access another tester's data outside an authorized security test.

## Enforcement

Report concerns privately to the repository owner. The owner may remove repository or beta access when necessary to protect contributors, testers, or user data.
